# VOX
> Vectorized Online Experiments

📚 See our [Wiki](https://github.com/ccp-eva/vox/wiki) for help and resources.


### Local Development
1. `git clone git@github.com:ccp-eva/vox.git`
1. `npm install`
2. `npm run dev`


### Depoly Application To A Server
1. `git clone git@github.com:ccp-eva/vox.git`
1. `npm install`
2. `npm run build`
3. Upload the contents within the `dist` folder to your web hoster.
